قالب ميديكال كير

مرحبا فى قالب ميديكال كير (الإصدار الأول)

ميديكال كير قالب احترافى لحجز عيادة أو دكتور فى مستشفى او عيادة من إختيارك ومتعدد الاستخدامات يتضمن العديد من الصفحات ويتميز بسهولة الاستخدام والتعديل عليه متوافق مع جميع لغات البرمجة والمكتبات ويتميز أيضاً بالتحديثات المستمرة.

مميزات القالب

  • متجاوب مع جميع الاجهزة   :   تبدو رائعة على أجهزة الكمبيوتر المكتبية وأجهزة الكمبيوتر المحمولة والأجهزة اللوحية والهواتف المحمولة. جربه على جهاز iPhone أو iPad الآن.
  • يدعم اللغة العربية   :   يتم دعم جميع العروض التوضيحية والصفحات الداخلية للتسوق من اليمين إلى اليسار.
  • 8+ صفحات Html   :   يحتوي ميديكال كير على أكثر من 8 صفحة html.
  • الموقع SASS   :   ميديكال كير HTML SASS
  • فحص W3   :   هذا القالب يحتوى على كود نظيف ويسهل التعديل عليه ، من خلال معايير W3
  • شرائح عرض للمنتجات   :   ميزة عرض المنتجات فى شكل شرائح
  • دعم متعدد المتصفحات   :   ميديكال كير لديه دعم المتصفحات مثل ( جوجل كروم، فايرفوكس، سفارى )
  • قابل للتخصيص بالكامل   :   يمكن تخصيص شكل معين من القالب لاحتوائه على العديد من التخصيصات.
  • بوتستراب اخر اصدار 5.1.3   :   ميديكال كير يحتوى على اخر اصدار من المكتبة الشهيرة بوتستراب 5.1.3
  • خطوط جوجل   :   ميديكال كير يحتوى على افضل الخطوط من جوجل ويمكن التعديل عليها بسهولة.
  • القائمة   :   ميديكال كير لديه قائمة مذهلة.

RTL - LTR

اذا كنت تريد تغيير اتجاهات الموقع من الشمال لليمين او العكس ماعليك سوى حذف استدعاء ملف rtl.css 

          
            <!-- Path File: (dist/css/rtl.css) -->
            <link href="dist/css/rtl.css" rel="stylesheet">

الخطوط

خط العربى : tajawal

خط الانجليزى: montserrat

لتغيير اللغه ستجد الفونت فى ملف _vars.scss فى المتغيرات التاليه :

--englishFont: 'Montserrat', sans-serif;
--arabicFont: 'Tajawal', sans-serif; 

          
            <!-- Path File: (scss/helper/_vars.scss) -->
            <link href="https://fonts.googleapis.com/css2?family=Montserrat:ital,wght@0,100;0,200;0,300;0,400;0,500;0,600;0,700;0,800;0,900;1,100;1,200;1,300;1,400;1,500;1,600;1,700;1,800;1,900&family=Tajawal:wght@200;300;400;500;700;800;900&display=swap" rel="stylesheet">

تغيير الألوان

    $colorOne: #3f4079;
    $colorTwo: #22c7b8;
    $colorThree: #4f6466;

نموذج البحث العام للموقع

  • يوجد تخصيص داخل ملف main.js لإعدادت نموذج البحث وشرحها كالأتى /

    Path File: dist/js/main.js

    حيث يوجد اربعه اعدادت للأربع مدخلات البحث 

          
            // 1. مدخل - تحديد التخصص
            $("#select2-specialize").select2();

            $("#select2-country").select2();

            // 3. مدخل - اختر المنطقة
            $("#select2-area").select2();

            // 4. مدخل - اختر العيادة
            $("#select2-clinics").select2();
          
            // اعداد عنوان توضيح المدخل ماذا يكتب هنا
            $("select").one("select2:open", function (e) {
              $(".select2-search__field").prop("placeholder", "أدخل ماتبحث عنه");
            });
